java 바이트 코드는 플랫폼에 독립적
JVM (자바가상머신) 은 플랫폼에 종속적

비트(bit)
Binary digit

컴파일
소스코드를 바이너리 파일(0,1)로 변환하는 과정

JVM (자바 가상 머신)
자바원시프로그램(java 확장자)를 컴파일하여 자바바이트코트 (class 확장자) 로 변경

JRE (자바 실행 환경)
JVM + 실행시키게 위해 필요한 도구

JDK (자바 개발 도구)
JRE + 필요한 도구 (문서, 컴파일러)

IDE (integrated development environment)
통합개발환경

println은 알아서 줄을 바꿔준다

java의 기본형 int, double
long은 L float은 f 로 표현

참조(reference)-name / 주소-0x0001

묵시적 (넓->좁) / 명시적 (좁->넓)

증감 연산자 (전위 ++i, 후위 i++)
비트연산자(~)

정수와 정수의 연산은 결과가 정수이다
즉 정수 나누기(/) 정수 결과의 소숫점 부분은 버려진다

java는 반복문에 이름을 붙여서 한번에 break 하는 방법이있음
